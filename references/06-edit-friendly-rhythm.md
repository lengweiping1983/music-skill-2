# 06 节奏可卡点

## 资料范围

本项使用 `source-bank-50.md` 的50篇来源，重点参考 S06-S07、S19、S27、S30、S40、S48-S50。核心问题：歌曲不是只给人听，还要给人剪视频、变装、转场、舞蹈和混剪。

## 50篇来源清单

本项来源清单见 [source-bank-50.md](source-bank-50.md) S01-S50；重点使用 TikTok 创作实践、groove/entrainment、短视频 BGM 结构、动态能量曲线和 Suno v5.5 资料。

## 共识结论

- 可卡点歌曲需要清楚重拍、停顿、drop、fill、转场点或 chant 落点。
- 鼓点不一定复杂，但必须让用户知道动作该落在哪。
- 变装、转场、慢动作、打拳、甩头、表情反转都需要明确时间点。
- 卡点不只靠鼓；人声断句、齐唱、sub drop、静音也能卡。
- 手机外放下节奏层必须清楚，低频不能糊成一团。

## 冲突观点

- 叙事民谣可以弱鼓，但仍要有可剪辑的呼吸点和句尾落点。
- 金属和工业可以密集，但必须给用户留下动作落点。

## 可执行 Suno 写法

`Style of Music` 写入：

```text
tight edit-friendly drums, clear downbeat, sudden stop before chorus, punchy transition hits, phone-speaker clear rhythm
```

`Lyrics` 写法：

```markdown
[Pre-Chorus | build | cut point]
(鼓点逐步收紧, 最后一拍突然停)
三 二 一

[Chorus | drop | edit point]
我把黑夜撞开
```

## 反例

- 全曲无停顿、无重拍、无转场。
- 鼓点很复杂但动作落点不清楚。
- 低频过糊，手机听不到节奏边界。

## 交付自检

- 是否有至少3个明确卡点？
- Chorus 前是否有停顿或转场提示？
- 节奏是否适合变装/转场/混剪之一？
- 手机外放是否仍能听见鼓点和句尾？
