# 12 预热-挑战-二创路径

## 资料范围

本项使用 `source-bank-50.md` 的50篇来源，重点参考 S01-S16、S40、S44-S46、S49-S50。核心问题：现代爆款常由片段预热、挑战模板、UGC扩散和完整版承接共同推动。

## 50篇来源清单

本项来源清单见 [source-bank-50.md](source-bank-50.md) S01-S50；重点使用 TikTok Sound Page、creator seeding、跨平台分发、AI artist persona、多版本和转化资料。

## 共识结论

- 创作阶段要先设计能被预热视频使用的最强10-20秒。
- 挑战模板需要明确动作、情绪或剧情反转，而不是只喊口号。
- 歌曲结构要支持 UGC：Hook Preview、Post-Chorus chant、可循环尾巴。
- 完整歌曲要承接短片段，避免用户点进来后失望。
- 人设或故事可以增加记忆，但不得写入 `suno.md` 的分析部分。

## 冲突观点

- 营销路径不是 `suno.md` 内容；Skill 只把路径反推成音乐结构。
- 挑战感强不等于廉价，要用具体动作和声音设计提升质感。

## 可执行 Suno 写法

`Lyrics` 结构：

```markdown
[Hook Preview | challenge phrase]
别眨眼
我就亮起来

[Post-Chorus | challenge chant]
亮起来 亮起来
(clap clap, group chant)
```

## 反例

- 在 `suno.md` 写“适合发起挑战”。
- 只有营销说明，没有可执行音乐动作。
- 完整副歌与预热片段声音完全不同。

## 交付自检

- 最强10-20秒是否清楚？
- 是否有可挑战动作或口号？
- 完整歌是否承接片段？
- 是否没有把营销计划写入 `suno.md`？
